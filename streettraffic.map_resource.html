get_map_tile_resource(location_data: tuple, location_type: str, zoom: int, img_size: int) → str[source]¶

inputs: location_data: tuple, zoom:int(zoom level), img_size:int(256 or 512), location_type: str

The location_type and corresponding location data is as follow location_type = “latlon”; location_data = (latitude, longitude) location_type = “colrow”; location_data = (column, row)

This function is uses location_data to generate (col, row), if location_type == “colrow”, we simply use the provided (col, row) and further use them to generate a url to get map_tile.

For map_tile resource details, refer to https://developer.here.com/rest-apis/documentation/enterprise-map-tile/topics/quick-start.html

return :str(url for map_tile)

get_traffic_json_resource(location_data: tuple, location_type: str, zoom: int) → str[source]¶

The location_type and corresponding location data is as follow location_type = “latlon”; location_data = (latitude, longitude) location_type = “colrow”; location_data = (column, row)

This function is uses location_data to generate (col, row), if location_type == “colrow”, we simply use the provided (col, row). Then we use (col, row) to further utilize self.get_quadkeys() to get a quad key, then generate a url to get traffic_json resource.

static produce_polygon(polygon_ordered_coordinates: typing.List, zoom: int, plot_polygon=True) → matplotlib.path.Path[source]¶

inputs: polygon_ordered_coordinates: List(an ordered list of points that composes a polygon) plot_polygon = True zoom: int example inputs: [[25.890099, -80.264561], [25.851873, -80.308744], [25.81123, -80.248538], [25.848596, -80.215765]], True

This function use matplotlib.path to create a Path/polygon object. Later we can use this Path/polygon object to invoke Path.contains_point() method. For more details on contains_point(), refer to https://stackoverflow.com/questions/21328854/shapely-and-matplotlib-point-in-polygon-not-accurate-with-geolocation

If plot_polygon == True, we use matplotlib to plot the polygon
